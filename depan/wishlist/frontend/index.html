Fitur Unggulan

Semua yang Anda butuhkan untuk mengelola wishlist

Create

Tambahkan barang baru dengan detail lengkap: nama, harga, kategori, dan deskripsi.

Read

Lihat semua wishlist dengan filter kategori, prioritas, dan pencarian cepat.

Update

Edit informasi barang kapan saja. Tandai sebagai terbeli dengan satu klik.

Hapus item yang tidak diinginkan. Bulk delete untuk item yang sudah terbeli.

Responsive

Tentang Aplikasi

Wishlist App adalah aplikasi full-stack yang dibuat untuk GDGoC Frontend Task. Aplikasi ini mendemonstrasikan kemampuan dalam membuat aplikasi web modern dengan fitur CRUD lengkap, desain responsif, dan backend API.

Tech Stack:

HTML5 CSS3 JavaScript ES6+ Node.js Express.js

API Endpoints

GET /api/v1/wishlist Get all items
GET /api/v1/wishlist/:id Get single item
POST /api/v1/wishlist Create item
PUT /api/v1/wishlist/:id Update item
PATCH /api/v1/wishlist/:id/toggle Toggle status
DELETE /api/v1/wishlist/:id Delete item
