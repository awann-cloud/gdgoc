DEPLOYMENT URL
==============

URL: https://[USERNAME].github.io/portfolio

OPSI DEPLOYMENT:

1. GitHub Pages (Recommended - Free)
   URL Format: https://[USERNAME].github.io/portfolio
   Cara Deploy:
   - Go to Repository Settings > Pages
   - Source: Deploy from branch
   - Branch: main / (root)
   - Save dan tunggu beberapa menit

2. Vercel (Free)
   URL Format: https://portfolio-[USERNAME].vercel.app
   Cara Deploy:
   - Import dari GitHub repository
   - Auto-deploy setiap push

3. Netlify (Free)
   URL Format: https://[USERNAME]-portfolio.netlify.app
   Cara Deploy:
   - Drag & drop folder atau connect GitHub
